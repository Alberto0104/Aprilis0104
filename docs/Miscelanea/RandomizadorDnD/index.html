Randomizador de Plantas

Este pequeño proyecto consiste en un randomizador para seleccionar plantas para las partidas de DnD, especialmente útil para aquellas clases como druida o explorador que quieran emplear dinámicas de búsqueda de plantas para aprovechar sus propiedades. En un principio fue diseñado para uso personal, pero puede servirle a cualquier persona.
