Randomizador de Plantas

Este pequeño proyecto consiste en un randomizador para seleccionar plantas para las partidas de DnD, especialmente útil para aquellas clases como druida o explorador que quieran emplear dinámicas de búsqueda de plantas para aprovechar sus propiedades. En un principio fue diseñado para uso personal, pero puede servirle a cualquier persona. Especial mención a NeurotoxicNihilist, usuario de Reddit y autor del documento en el que se basa este pequeño proyecto y se tomó el tiempo y el esfuerzo en documentar de manera tan detallada todo. Podéis encontrar su trabajo aquí.
